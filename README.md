
# Sistema Bancario - Backend

Sistema de gestión bancaria con microservicios usando Spring Boot y PostgreSQL.

## Arquitectura

- **ms-cliente-persona** (Puerto 7085): Gestión de Clientes y Cuentas
- **ms-cuenta-movimiento** (Puerto 7086): Gestión de Movimientos y Reportes
- **PostgreSQL**: Base de datos
- **Angular**: Ultima ver.
  
## Tecnologías

- Java 17
- Spring Boot 3.2.0
- Gradle (Groovy)
- PostgreSQL
- Docker & Docker Compose
- JPA/Hibernate
- Lombok
- iText7 (generación PDF)

## Requisitos Previos

- Docker
- Docker Compose
- Java 17 (solo para desarrollo local)
